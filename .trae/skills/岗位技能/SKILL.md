# 岗位技能

## 功能

当用户提到"岗位技能"时，提供相关的岗位技能信息和建议，包括：
- 不同岗位的核心技能要求
- 技能提升建议和方法
- 技能评估和发展路径
- 行业趋势和技能需求变化

## 使用方法

当用户提到"岗位技能"时，调用此技能，根据用户的具体需求提供相关的岗位技能信息和建议。

## 内容要求

1. **实用性**：提供具体、可操作的技能建议
2. **针对性**：根据不同岗位和行业提供相关技能信息
3. **前瞻性**：包含行业发展趋势和未来技能需求
4. **全面性**：覆盖硬技能和软技能

## 示例

**输入**：岗位技能

**输出**：
不同岗位的核心技能需求各有不同，例如：
- 技术岗：编程能力、问题解决、系统设计
- 营销岗：市场分析、内容创作、数据分析
- 管理岗：领导力、沟通协调、战略规划

建议通过项目实践、在线课程和行业交流持续提升技能，同时关注行业趋势，培养未来所需的新兴技能。
