# 小红书文案生成

## 功能

根据用户提供的关键词，生成小红书风格的文案，特点包括：
- 字数小于100字
- 包含丰富的表情符号
- 符合小红书平台的语言风格
- 亲切、活泼、有感染力

## 使用方法

当用户输入关键词并提到"小红书文案"时，调用此技能，根据关键词生成符合要求的小红书风格文案。

## 生成要求

1. **字数限制**：文案长度控制在100字以内
2. **表情使用**：适当添加表情符号，增强文案的生动性
3. **风格特点**：
   - 口语化、亲切自然
   - 突出产品/主题的亮点
   - 具有个人体验感
   - 引导互动（如点赞、收藏、评论）
